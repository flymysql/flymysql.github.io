大学生，你凭什么创业？

发表于 2018-04-21 | 更新于: 2018-05-14 | 分类于 随笔 | | 热度: 2℃
字数统计: 1.8k | 阅读时长 ≈ 6

所以，就不要在做着那些不读书混社会的学生当了大老板，三好学生毕业了给你打工的梦。

前几日看到朋友圈有朋友在转发学校创新创业比赛的消息，几个其他院的学生在找计算机专业的同学做个app。每年都这个样子，仿佛现在大学生对创新创业的理解就是，几个人聚在一起想个idea，然后再找人做个app。和以前大学生动不动就开奶茶店创业似乎异曲同工，实话说开奶茶店都比找人瞎做app实在。

开奶茶店的大学生还舍得投入两三万踏踏实实干，然而有些人以为找个免费劳动力就能搭个app。幻想着未来毕业要出去创业，抱怨着读书没什么用，反正要出去创业，靠实力的。实力？不知道这些人对实力的理解是不是他们所理解的那所谓“非常重要的交际能力”。

创业固然好，国家鼓励，学校支持。当然了，越多的大学生创业，就会提供越多的工作岗位，学校的就业率也会提升。只要你毕业的时候在就业去向那一栏填上“创业”，就为学校的就业率增加一点贡献，至于你之后的死活，谁理你？
