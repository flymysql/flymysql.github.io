百度网盘不限速下载，你的网速有多快，下载速度就有多快！！

  小鸡出品，必属精品  


小鸡今天逛Github发现了一个好东西

BaiduPC-Go


一个百度网盘不限速的下载器

电脑端需要终端操作

小鸡去试水了

感觉很棒哦！



不过需要会一点终端命令行操作

但是对于8012年的年轻人

小鸡觉得这点操作应该自己百度就会了




那么全速的下载速度如何呢？

下面是小鸡的测试






2.5兆每秒

这还是因为小鸡宿舍的网太渣

小鸡平常上宿舍网的速度都没这么快哦


所以只要你的网速有多快

网盘的下载速度就有多快


那么要去哪里下载使用的？

尊重原作者，这里给出作者github地址


复制到浏览器打开

https://github.com/iikira/BaiduPCS-Go/releases



具体使用方法看github上的说明

当然这么好的东西小鸡也把它fork到自己的github里

https://github.com/flymysql/BaiduPCS-Go

（不懂一定要看说明！）



那么，该怎么用呢？


第一
文件下载下来建议存放在英文目录下。


第二
右键打开我的电脑-属性-高级-环境变量-系统变量 -Path



新建填入BaiduPCS-Go 的存放路径。



第三
开始菜单，输入 cmd，右键选择以管理员身份运行。
这点要注意，不然会被系统误判为病毒



当然你也可以 win+R 运行 cmd 命令行，但这种有时候在执行命令时会报错弹出'BaiduPCS-Go' 不是内部或外部命令，也不是可运行的程序。所以建议大家还是以管理员身份运行。

第四
输入（具体看下载的.exe文件名，可能不一样） baidupcs-go，回车执行。

名字比较长小鸡建议去文件夹里把baidupcs-go.exe的名字改掉
比如baidu.exe，这样输入比较方便，可以直接在终端输入baidu就可以了





第五
登陆百度云账号

在命令行窗口输入 login

根据提示输入账号密码


第六
找一个文件下载来试试


如果你需要下载别人分享的文件，首先要保存在自己网盘内，我这里以百度盘根目录下的一个文件进行测试，如果你的文件存在目录下，记得要复制完整的路径以及文件名（包括扩展名）


在命令行窗口输入下载命令：d  下载的文件名 ，格式为 d（空格）（文件路径，若有）/（文件全名）。回车开启下载。



当你看到这样的速度就会觉得，上面略繁琐的操作都是值得的。刚开启速度略慢，但很快就会飙升到满速下载，此外，需要下载的文件名不要含有空格、小括号、 中括号或者是特殊字符，不能被识别。

更多使用相关说明，请大家参看软件在 Github 上的项目页面。

10.文件下载后会存放在 baidupcs-go 的路径下。



提示：上述操作只需设置一遍就好，再次执行 baidupcs-go，就无需再次登陆，直接输入要下载的文件路径及名称就好，其实也非常方便，关键是下载速度是真快呀！


对于计算机小白可能有点繁琐，但是这个下载速度还是值得大家去折腾的。
